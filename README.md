# rotatingImage
## 프로젝트 소개
이미지 클릭시 이미지가 돌아가는 웹사이트

### 사용 기술
HTML, CSS, JS

## 협업 진행 방식
git flow 방식 사용
![git flow](./images/gitflow_image.png)
dev, feature 브랜치 사용
### commit 규칙
동사형 영어( Feat, Add) 사용

## 어려웠던 점
- 김은지 : 자꾸 현재 브랜치가 어디인지 까먹게 된다. 생각보다 역할 나누기가 어렵다.
- 박희범 : Github PR 페이지에서 merge branch 하는 부분이 눈에 잘 안 익는다. 실수하기 쉬운 부분인 거 같다.
